Election System Instructions

  1. Add an election
  2. Add positions for this election
  3. For each position, add candidates (addable in the admin page for each position)

Once an election is setup, people can proxy vote by visiting the elections page

Things to note:

  • For those organising the election, there is a summary page accessible from the elections page which provides a list of completed proxy votes
  • There is also a page that lists the university ids of all our active members, with lines through those who have already proxy voted
  • There is also a page that presents a printable ballot form
  • Guests may not proxy vote
